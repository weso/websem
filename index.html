Context

These web pages are developed for two courses that are part of the Master in Web Engineering at the School of Computer Science Engineering.

Although this web site is in English, the subject is taught in Spanish, so most of the slides are in Spanish.

The courses are Semantic web (Web Semántica) and New advances in Semantic web (Nuevos avances en Web Semántica). We decided to keep a single web page for both courses as they share some teaching material

Course editions

The course started in 2007 but we were using University of Oviedo's virtual campus infrastructure to store the material. In 2020, we decided to create this web page to allocate public resources.
